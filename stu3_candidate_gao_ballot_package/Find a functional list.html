descriptionThis operation allows a client to find an identified list for a particular function by its function. The operation takes two parameters, the identity of a patient, and the name of a functional list. The list of defined functional lists can be found at [Current Resource Lists](lifecycle.html#lists). If the system is able to locate a list that serves the identified purpose, it returns it as the body of the response with a 200 OK status.
